
Inventaire cabanon — PWA prête à l'emploi (FULLSCREEN)

Contenu:
- index.html
- manifest.webmanifest (display: fullscreen)
- sw.js (service worker)
- js/app.js
- css/style.css
- data/catalogs/*.json (Hager, Legrand, Schneider, Sonepar, Wurth)
- icons/icon-192.png, icons/icon-512.png

Instructions rapides :
1) Upload all files to GitHub repo root and enable Pages.
2) Use PWABuilder to build Android package.
3) Or run local HTTP server and open index.html then Add to Home (Chrome) -> it will open fullscreen.
